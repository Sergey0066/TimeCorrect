# TimeCorrect

Если на Windows время отстаёт/спешит на 1–3 минуты или после включения ПК время становится неверным, программа `TimeCorrect` принудительно синхронизирует системное время через службу Windows Time (`w32time`) и NTP (команды `w32tm`).

## Запуск

1) Скачай последнюю версию: [`TimeCorrect.exe`](https://github.com/Sergey0066/TimeCorrect/releases)
2) Запусти `TimeCorrect.exe`
3) После запуска:
   - время синхронизируется по сети.
   - создаётся автозапуск — при запуске ПК время синхронизируется сразу.

Если хочешь удалить, тогда заходишь в пуск - TimeCorrect - пкм - удалить.

### Возможности

- Синхронизация времени по NTP (по умолчанию `pool.ntp.org`)
- Работает на Windows 10/11
- Скрытая работа, без отображения консоли
- Автозапуск при старте Windows
